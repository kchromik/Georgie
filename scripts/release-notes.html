What's new

  • Throw widgets like native PiP: flick a widget toward a screen edge or corner and it glides there, keeping a small margin from the edge.
  • Edge magnetism reworked: snapping now applies reliably when you drop a widget near an edge; deliberately shoving it past the edge parks it with a grab handle still visible.
  • Session restore fixed: quitting the app no longer wipes the saved widget session; widgets lost offscreen are rescued back into view on restart.
  • Window Mirror: the window picker now only lists real application windows — no more Dock, wallpaper, or system overlays.
  • The video widget no longer offers its own picture-in-picture button.
