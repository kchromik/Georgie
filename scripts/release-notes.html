What's new

  • Window Mirror: capture any other app window as a live floating widget.
  • Markdown notes: the note widget now renders markdown — headings, lists, clickable checklists, quotes, and code blocks.
  • Web fullscreen, fixed: video fullscreen (e.g. YouTube) now stays inside the widget instead of taking over the screen.
  • Edge parking: shove widgets past the screen edge to tuck them away; snap to edges now applies reliably on drop.
  • Widgets restored at login can now be closed normally, and closing a web widget reliably stops audio playback.
  • Removed the Clock/Timer and System Monitor widgets and the camera shape options.
